# MOTION//PULSE

웹캠 전신 인식 기반 모션 아케이드입니다. 네 가지 게임을 선택할 수 있으며 영상은 브라우저 안에서만 처리되고 저장하거나 서버로 전송하지 않습니다.

## 게임

- **NEON TOUCH**: 공간에 등장하는 코어를 손으로 터치하고 스쿼트 게이트를 통과하는 60초 게임
- **BEAT BOX**: 좌우 손을 번갈아 빠르게 뻗는 45초 리듬 펀치 게임
- **SQUAT TUNNEL**: 앉기와 완전히 일어서기를 교대로 판정하는 45초 반복 게임
- **POSE FLASH**: 양팔 들기, T 자세, 좌우 기울이기, 스쿼트를 따라 하는 60초 전신 게임

## 실행

1. `start-game.bat`를 더블 클릭합니다.
2. 브라우저에서 **카메라로 시작**을 누릅니다.
3. 카메라 권한을 허용하고, 어깨부터 무릎까지 보이도록 1.5~2m 뒤에 섭니다.

인터넷 연결은 최초 포즈 인식 모듈을 불러올 때 필요합니다. 카메라 없이 확인하려면 **카메라 없이 체험**을 선택하고, 코어는 마우스 또는 `Space` 키로 터치하며 스쿼트 게이트는 `S` 키로 통과합니다.

## 판정 방식

- 일반 코어: 양쪽 손목 중 하나가 코어 반경 안에 약 0.15초 머물면 성공
- 왼손/오른손 코어: 지정된 손목만 판정
- 스쿼트 게이트: 어깨·엉덩이·무릎의 상대 높이와 무릎 굽힘 각도를 함께 판정
- 전신 포즈: 손목·어깨·엉덩이의 상대 위치를 약 0.42초 유지하면 성공
- 연속 성공 시 콤보 보너스가 최대 +200점까지 증가

조명이 밝고 몸과 배경이 잘 구분될수록 인식이 안정적입니다.

## 모바일·태블릿

- 휴대폰 세로, 태블릿 세로와 가로 화면에 맞춘 별도 반응형 배치
- 노치와 홈 표시줄을 피하는 안전영역 적용
- 화면 방향에 맞춰 전면 카메라의 세로·가로 해상도를 자동 요청
- 터치 버튼 최소 높이 48px 및 화면 가로 넘침 방지
- 게임별 생성 선화 4장과 아레나 배경을 모바일용 WebP로 압축
