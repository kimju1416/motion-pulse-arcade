# MOTION//PULSE

웹캠 전신 인식 기반 모션 아케이드입니다. 네 가지 게임을 선택할 수 있으며 영상은 브라우저 안에서만 처리되고 저장하거나 서버로 전송하지 않습니다.

## 게임

- **ORDER TOUCH**: 여러 숫자 중 1번부터 순서대로 손으로 터치하는 60초 게임
- **MATH HIT**: 화면의 덧셈을 계산하고 세 개의 답 중 정답을 터치하는 60초 게임
- **SQUAT COUNT**: 실제 엉덩이·무릎·발목 각도로 앉기와 일어서기를 판정하는 45초 게임
- **JUMPING JACK**: 팔과 다리를 모두 벌렸다가 모두 모아야 1회로 세는 45초 게임

## 실행

1. `start-game.bat`를 더블 클릭합니다.
2. 브라우저에서 **카메라로 시작**을 누릅니다.
3. 카메라 권한을 허용하고, 화면 안내에 맞춰 섭니다. 스쿼트와 팔벌려뛰기는 머리부터 발목까지 들어오도록 약 2~3m 거리를 확보하세요.

인터넷 연결은 최초 포즈 인식 모듈을 불러올 때 필요합니다. 카메라 없이 확인하려면 **카메라 없이 체험**을 선택하세요. 순서·덧셈 게임은 타깃 클릭 또는 `Enter`, 스쿼트는 `S`, 팔벌려뛰기는 `J`로 조작합니다.

## 판정 방식

- 카메라 영상: 화면을 확대해 자르지 않는 contain 방식으로 표시하고 관절·타깃도 같은 좌표계를 사용
- 숫자 터치: 75ms 이상 타깃 안에 들어오고, 손을 빼야 다음 입력이 활성화
- 관절 표시: 판정에 쓰는 관절을 네온 노드로 표시하고 손목 궤적·무릎 각도·팔/다리 OPEN 상태를 실시간 강조
- 성공 피드백: 맞힌 위치에 체크 링, 정답 식, 획득 점수를 표시하고 오답 위치에는 빨간 ×를 표시
- 스쿼트: 실제 발목 관절을 포함한 양쪽 무릎 각도와 엉덩이 하강을 함께 판정
- 팔벌려뛰기: 양팔을 머리 위로 올리고 양발도 벌린 뒤 차렷 자세로 돌아오면 성공
- 추적 상실: 몸이 잘리면 점수와 제한시간을 자동으로 일시정지
- 연속 성공 시 콤보 보너스가 최대 +200점까지 증가

조명이 밝고 몸과 배경이 잘 구분될수록 인식이 안정적입니다.

## 모바일·태블릿

- 휴대폰 세로, 태블릿 세로와 가로 화면에 맞춘 별도 반응형 배치
- 노치와 홈 표시줄을 피하는 안전영역 적용
- 화면 방향에 맞춰 전면 카메라의 세로·가로 해상도를 자동 요청
- 화면 회전 시 동작을 잠시 멈추고 몸 위치를 다시 확인
- 터치 버튼 최소 높이 48px 및 화면 가로 넘침 방지
- 게임별 생성 선화 4장과 아레나 배경을 모바일용 WebP로 압축
