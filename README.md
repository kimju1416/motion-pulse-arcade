# 말랑잉글리시

카메라 앞에서 몸을 움직이며 영어를 익히는 유아용 웹 게임입니다. 영상은 브라우저 안에서만 동작 인식에 사용되며 저장하거나 서버로 전송하지 않습니다.

## 영어 놀이 4종

- **철자 톡톡 · SPELL POP**: 그림과 한글 힌트를 보고 3~4글자 영어 철자를 차례대로 손으로 터치합니다.
- **그림 단어 찾기 · PICTURE PICK**: 한국어 뜻을 듣고 알맞은 영어 단어를 두 개의 큰 카드 중에서 고릅니다.
- **영어 동작 미션 · MOVE & SAY**: 서로 다른 영어 동작 6개를 하나씩 따라 하며 `1/6`부터 `6/6`까지 미션을 완성합니다.
- **별 모양 쭉쭉 · STAR SHAPE**: 뛰지 않고 팔과 다리를 별 모양으로 천천히 벌렸다 모읍니다.

단어 덱은 한 바퀴를 모두 마칠 때까지 같은 단어가 반복되지 않습니다. 문제는 먼저 한국어로 들려주고, 정답을 맞힌 뒤 영어 단어와 발음을 보여 줍니다. 플레이 화면의 **다시 듣기** 버튼으로 현재 안내를 반복할 수 있습니다. 오답에는 점수를 빼거나 낮은 등급을 주지 않고 부드러운 힌트만 제공합니다.

## 동작 놀이 안전 약속

- 주변을 두 걸음 비우고 미끄럽지 않은 바닥에서 놀이합니다.
- 휴대폰이나 태블릿은 넘어지지 않게 단단히 세웁니다.
- 유아는 보호자와 함께하며, 아프거나 어지러우면 바로 멈춥니다.
- 별 모양 놀이는 높이 뛰지 않아도 됩니다. 팔과 다리를 천천히 벌리고 모으면 인식됩니다.

## 실행

1. `start-game.bat`를 더블 클릭합니다.
2. 브라우저에서 **카메라로 시작!**을 누릅니다.
3. 카메라 권한을 허용하고 화면 안내에 맞춰 섭니다.

카메라 없이 확인하려면 **먼저 구경하기**를 선택하세요. 체험 모드에서는 화면 타깃 클릭 또는 `Enter`, 영어 동작 미션은 `S`, 별 모양 놀이는 `J`로 조작합니다.

## 인식과 반응성

- 상반신 단어 게임은 머리·어깨와 한쪽 손만 보여도 시작합니다.
- 카메라는 최대 640×480, 동작 분석은 약 17회/초로 제한해 모바일 부담을 줄였습니다.
- 손 선택은 큰 카드 안에 약 0.14초 머무르면 동작하며, 손을 가운데로 되돌린 뒤 다음 입력을 받습니다.
- 영어 동작 미션과 별 모양 놀이는 무릎·발목을 포함한 전신 관절을 사용합니다.
- 카메라 영상은 자르지 않는 contain 방식으로 표시하고 관절과 타깃도 같은 좌표계를 사용합니다.
- 추적이 오래 끊기면 놀이 시간을 자동으로 멈추고 입력을 안전하게 다시 준비하며, 화면 회전 후에도 점수와 진행 상황을 유지합니다.

## 유아용 UI

- 밝은 하늘·크림·노랑·코랄·민트 색상과 둥근 카드 디자인
- 모든 주요 버튼 최소 48px, 큰 선택 타깃과 짧은 한글 안내
- 생성형 이미지로 만든 곰·토끼·사자·펭귄 게임 카드와 파스텔 놀이터 배경
- 감점·C등급·빨간 실패 표시 대신 별 스티커와 “한 번 더” 피드백
- 한국어 문제 안내, 정답 뒤 영어 발음, 시각 자막과 항상 보이는 다시 듣기 버튼
- 휴대폰 세로·가로, 태블릿 세로·가로, 노치·홈 표시줄 안전영역 대응
- `prefers-reduced-motion` 환경에서는 팝·전환 애니메이션을 최소화

인터넷 연결은 최초 포즈 인식 모듈과 웹 글꼴을 불러올 때 필요합니다. 조명이 밝고 몸과 배경이 잘 구분될수록 인식이 안정적입니다.
